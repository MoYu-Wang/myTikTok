### 请求参数和响应参数都是json形式

### 后端接口说明
/user/register: 注册用户
/user/login: 用户登录
/user/update: 用户修改信息
/user/info: 获取用户信息
/user/base: 获取本用户基本信息
/user/forgetpwd: 找回密码
/user/delete: 用户注销
/user/updatetoken: 更新用户登录信息
/user/works: 用户作品
/user/favorite: 用户点赞视频列表
/user/history: 用户观看历史记录

/video/top: 获取热点视频
/video/care: 获取关注视频
/video/referee: 获取推荐视频
/video/getsign: 获取上传签名
/video/upload: 上传视频
/video/commit: 评论视频
/video/operate: 划走视频后对视频的操作

(以下暂未实现)
/broadcast: 直播
/shopping: 商城
